5. Key Findings

  1. All five tracks produce meaningful score spreads — skills don't cluster at the same level
  2. Skill value is model-dependent — same skill: Δ=+2 on pro, Δ=+24 on flash
  3. Output format constraints hurt creative tasks (Track B Δ=-5.5) but help structured ones (+3.0)
  4. Only actual format skills produce positive Track C deltas — general skills forced into format rubric score negative
  5. Judge JSON parsing produces ~15% data loss — known limitation, tracked in repo
